Quiero que generes una interfaz completa en HTML, CSS y JavaScript puro basada en estas pantallas de una tienda minimalista.

Actúa con una mentalidad de diseño inspirada en Steve Jobs y Jony Ive:
- simplicidad
- elegancia
- claridad visual
- enfoque obsesivo en la experiencia del usuario
- diseño minimalista pero funcional

## Objetivo
Construir una pequeña tienda web con navegación entre pantallas, replicando la experiencia visual y de interacción de las imágenes de referencia.

## Pantallas que necesito
1. Pantalla de búsqueda de productos
   - barra de búsqueda superior
   - sugerencias en chips o botones redondeados
   - sección de tendencias
   - lista de productos con imagen, nombre y breve descripción
   - barra de navegación inferior

2. Pantalla de catálogo por defecto
   - encabezado limpio con logo/nombre y buscador
   - título principal
   - listado vertical de productos
   - cada producto debe mostrar:
     - imagen
     - nombre
     - descripción
     - precio
   - barra de navegación inferior

3. Pantalla de detalle de producto en modal
   - al hacer clic en un producto, se debe abrir un modal grande o panel deslizante
   - el modal debe mostrar:
     - imagen grande del producto
     - nombre
     - precio
     - descripción
     - botón de “Añadir al carrito”
     - acciones visuales como favorito, compartir o zoom
   - debe poder cerrarse fácilmente

## Requisitos técnicos
- Usa solo HTML, CSS y JavaScript puro
- No uses frameworks ni librerías externas
- El código debe estar separado y organizado en:
  - index.html
  - styles.css
  - script.js

## Organización del HTML
Quiero que el HTML esté bien dividido en componentes o secciones semánticas, por ejemplo:
- header
- search-section
- suggestions-section
- trends-section
- product-list
- product-card
- modal
- bottom-nav

Aunque todo esté en un solo archivo HTML, quiero que la estructura sea modular, limpia y fácil de entender.

## Navegación e interacción
La app debe permitir:
- navegar visualmente entre la vista de búsqueda y la vista principal
- abrir el detalle del producto en un modal al hacer clic
- cerrar el modal
- simular una experiencia fluida y moderna

## Estilo visual
Quiero un diseño:
- minimalista
- moderno
- limpio
- elegante
- con mucho aire visual
- bordes redondeados
- tipografía clara
- buena jerarquía visual
- inspirado en productos premium

## Importante
Estoy aprendiendo programación, así que necesito que:
- el código esté muy bien documentado
- expliques el HTML, el CSS y el JavaScript en español
- uses comentarios claros dentro del código
- nombres variables y clases de forma comprensible
- la lógica sea sencilla de seguir
- expliques al final cómo funciona cada parte
- expliques cómo conectar el HTML con el CSS y JS
- expliques cómo podría modificar productos, textos e imágenes más adelante

## Resultado esperado
Entrégame:
1. El archivo HTML completo
2. El archivo CSS completo
3. El archivo JS completo
4. Una explicación en español, paso a paso, de cómo funciona todo
5. Buenas prácticas básicas para alguien que está empezando

Toma como referencia las imágenes compartidas y replica la estructura visual y la navegación de la forma más fiel posible.
